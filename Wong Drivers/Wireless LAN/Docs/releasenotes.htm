Supported Adapters

This version of Intel® PROSet/Wireless WiFi Software is compatible with the adapters listed below. Note that newer features provided in this software release are generally not supported on older generations of wireless adapters.

The following adapters are supported in Windows* 10:

  • Intel® Wi-Fi 6 AX201
  • Intel® Wi-Fi 6 AX200
  • Intel® Wireless-AC 9560
  • Intel® Wireless-AC 9462
  • Intel® Wireless-AC 9461
  • Intel® Wireless-AC 9260
  • Intel® Dual Band Wireless-AC 8265
  • Intel® Dual Band Wireless-AC 8260
  • Intel® Dual Band Wireless-AC 3168
  • Intel® Dual Band Wireless-AC 7265
  • Intel® Dual Band Wireless-N 7265
  • Intel® Wireless-N 7265
  • Intel® Dual Band Wireless-AC 3165

Operating Systems

This version of Intel® PROSet/Wireless WiFi Software runs on Microsoft Windows* 10 only.

Viewing User Documentation

To access user documentation, install the Intel® PROSet/Wireless WiFi Software and open Index.htm that has been copied to C:\Program Files\Intel\WiFi\Docs.

Upgrading Software from Previous Versions

When you run the Intel® PROSet/Wireless WiFi Software installation program, it replaces the previous version of Intel® PROSet/Wireless WiFi Software and/or WiFi driver with the current version.
