使用 WiFi 網路卡，您可以存取 WiFi 網路、共用檔案或印表機，或共用網際網路連線。您可以在家裡或辦公室透過 WiFi 網路使用這些功能。此 WiFi 網路解決方案專為家用與商用設計。隨著網路需要的增加與變更，可以新增其他用戶與功能。

根據您的 Intel WiFi 介面卡的型號，您的介面卡可能與 802.11a、802.11b、802.11g、802.11n、802.11ac 和 802.11ax 無線裝置標準相容。網路卡能以 5GHz 或 2.4GHz 頻率運行。您現在可以將電腦連接到在大型或小型環境中有多個存取點的高速網路。您的 WiFi 介面卡會根據存取點位置和訊號強度保持自動的資料速率控制，以盡可能達成最快的連線。

適用於所有使用者或經銷商的重要通知：

Intel 無線 LAN 介面卡已經過工程設計、製造、測試和品質檢查，以確保其符合所在地區所有必須的當地和政府法規機構的要求，在這些地區介面卡要經過指派和/或標記才能進入。由於無線 LAN 通常是未經授權的裝置，與雷達、衛星及其他授權和未經授權的裝置共用頻譜，所以有時需要動態偵測、避免和限制使用以避免與這些裝置干擾。在許多情況下，在授予使用產品的認證或核准前，Intel 都必須提供測試資料以證明產品在當地是否遵循該區域及政府規範。Intel 的無線 LAN 的 EEPROM、韌體和軟體驅動程式，均設計成能夠精細地控制影響無線操作的參數，並確保電磁相容性 (EMC)。這些參數包括但不限於 RF 功率、頻譜的使用、頻道掃描和人體暴露。
