Ρυθμίσεις προσαρμογέα

Η καρτέλα Για προχωρημένους εμφανίζει τις ιδιότητες συσκευής για τον προσαρμογέα WiFi που βρίσκεται εγκατεστημένος στον υπολογιστή σας.

Τρόπος πρόσβασης

Κάντε διπλό κλικ στον προσαρμογέα Intel WiFi στην ενότητα Προσαρμογείς δικτύου στη Διαχείριση Συσκευών και επιλέξτε την καρτέλα Για προχωρημένους.

Μπορείτε να βρείτε περιγραφή των ρυθμίσεων προσαρμογέα WiFi στην καρτέλα Για προχωρημένους, εδώ:

https://www.intel.com/content/www/us/en/support/articles/000005585/network-and-i-o/wireless-networking.html
