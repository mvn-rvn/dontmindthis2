Adapter Settings

The Advanced tab displays the device properties for the WiFi adapter installed on your computer.

How to Access

Double-click on the Intel WiFi adapter in the Network adapters section of the Device Manager and select the Advanced tab.

A description of the WiFi adapter settings on the Advanced tab can be found here:

https://www.intel.com/content/www/us/en/support/articles/000005585/network-and-i-o/wireless-networking.html
